Installation & Deployment

Chrome Extension

Install as browser extension for quick access

Installation Steps:

  1. Click "Add to Chrome" button above
  2. Allow popup if blocked by browser
  3. Click "Add Extension" in the popup
  4. Pin the extension for easy access

Desktop Widget

Add to desktop as Progressive Web App

Installation Steps:

  1. Click "Add to Desktop" button above
  2. Follow your browser's installation prompts
  3. App will be added to your desktop/app menu
  4. Works offline once installed

CLI Tool

Command line interface for developers

Installation Steps:

  1. Download the CLI tool for your platform
  2. Extract the archive to a folder
  3. Add the folder to your system PATH
  4. Run locksmith --help to get started
